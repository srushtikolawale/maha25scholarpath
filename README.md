# 🎓 Maha25ScholarPath

Maha25ScholarPath is a Flask-based scholarship guidance platform designed for students in Maharashtra aged 25 years or below. It helps users discover scholarships based on eligibility criteria such as caste, gender, annual income, education level, and age.

---

## 🚀 Features

- 👤 User Registration & Login
- 🔐 Email OTP Verification
- 🌐 English & Marathi Language Support
- 🔍 Scholarship Search & Filtering
- 📊 Eligibility-based Recommendations
- 💾 SQLite Database
- 📱 Responsive User Interface

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/srushtikolawale/maha25scholarpath.git
```

Move into the project directory:

```bash
cd maha25scholarpath
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
